# Markdown Lab

A minimalist, high-performance Markdown editor built with React, TypeScript, and Ant Design.

## Features

- **Split-Pane Layout**: Real-time editing and previewing.
- **Synchronized Scrolling**: Scroll one side, and the other follows automatically.
- **Dark/Light Mode**: Toggle between themes with persistence.
- **GitHub Flavored Markdown**: Support for tables, task lists, and code highlighting.
- **Responsive Design**: Works on desktop and mobile.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Tech Stack

- **Framework**: React 18 + Vite
- **Language**: TypeScript
- **UI Library**: Ant Design 5.x
- **Markdown Engine**: react-markdown + remark-gfm + rehype-highlight
- **Styling**: CSS Modules / Global CSS
